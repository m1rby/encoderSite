Detailed Description

Главный файл программы, содержащий реализацию логики работы с файлами и многозадачности.

Этот файл включает в себя основную логику программы. Он использует функции, объявленные в заголовочном файле main.h, для чтения и записи файлов, шифрования данных, вычисления контрольных сумм и многозадачности. Все операции выполняются с использованием стандартных библиотек C++ для обработки файлов и потоков.

Основные этапы работы программы:

  1. Чтение данных из файла с использованием многозадачности.
  2. Шифрование данных с использованием гамма-шифрования.
  3. Сохранение зашифрованных данных в новый файл с применением параллельной записи.
  4. Обработка ошибок и синхронизация потоков.
